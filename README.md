# ATM_Simulator
This project is a console-based banking application developed in Python that simulates basic ATM operations. It enables users to perform essential banking tasks such as depositing money, withdrawing cash, and checking their account balance through an interactive menu-driven interface.

The system enforces a daily transaction limit of three operations to mimic real-world banking constraints. It also ensures that all transactions are made in multiples of 100 and restricts withdrawals to a maximum limit of ₹20,000 per transaction. For withdrawals and deposits, the application provides a breakdown of currency denominations (₹500, ₹200, and ₹100), similar to how ATMs dispense cash.

The program uses class variables to maintain a shared account balance and transaction count across operations. It focuses on fundamental programming concepts such as conditional statements, loops, user input handling, and object-oriented programming in Python.

This project is ideal for beginners to understand how basic banking logic works and how real-world constraints can be implemented in code.
